## 문제점 / 고민한점
- 일부러 git 브랜치 간에 충돌을 내보고 싶었다. 그렇게 계속해도 충돌이 나지 않았다.

- 어떤 협업 툴을 사용하고 어떻게 협업을 해야 될 지를 많이 고민헀다. 물론 첫 술에 배부를 수는 없지만 계속 반복해보면서 익혀야 부분인 것 같다.
